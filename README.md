# Re:Bumi — Sustainable Footwear Export Landing Page

A premium B2B export landing page for **Re:Bumi Recycled Sandals by Pijak Bumi**, targeting wholesale buyers, importers, and distributors in Germany and the Netherlands.

## Key Technologies

- **TanStack Start** (React SSR framework)
- **Tailwind CSS v4** — utility-first styling
- **Framer Motion** — page and scroll animations
- **Netlify Forms** — serverless form submission handling
- **Google Fonts** — Playfair Display + Inter

## Running Locally

```bash
npm install
npm run dev
# Opens at http://localhost:3000
```

## Deploying

Push to your connected Git repository. Netlify auto-deploys on push.

> Note: Netlify Forms only works on deployed previews/production — not local dev.
